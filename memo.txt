BUYO
3ヶ月
PHP,JavaScript,Sass,SQLite,XServer
サークルで使っている部室の使用予約ができるサイトです。
今までメーリングリストアプリ内のチャットで行なっていた手間のかかる業務を、Webサイトに
アクセスして操作するだけ、ログイン不要という、最低限のフローで済ませられる
ようにすることを意識しました。また、予約･変更･削除時にはLINEの公式アカウントから
通知がくるようにもなっています。
デザインはいつかちゃんと考え直そうと思います。


Time Table Maker
2週間ほど
Sass,JavaScript,jQuery
サークルでのライブの際に、タイムテーブルや時間を見やすく表示するために作りました。
メモ帳のスクリーンショットで当日に共有されていたタイムテーブルは、オフライン環境で行なっていた
ライブ中にアクセスしても見られなかったため、完全オフラインで動作することを想定して作りました。


Oh-o!Meiji Server Error Bot
1週間
PHP,Cron
オンライン授業が始まった際、大学ポータルサイトがよくサーバーダウンしていたため、
TwitterBotの勉強も兼ねて作りました。1分ごとにCronでサーバーの状態をチェックし、
メンテナンス時、サーバーエラー時にツイートで教えてくれます。

image PDF Converter
2日
JavaScript,PDFMake(ライブラリ)
オンライン授業で「写真を撮ってPDFにして提出」という課題が出始め、「この写真PDFにして！」
とよく頼んできていた友達に向けて作りました。
github.ioで公開したかったため、クライアントサイドのみで動作するPDFMakeというJavaScript
ライブラリを使用しました。

Vue Gamble Calculator
3日
Vue.jsに慣れるために初めて作ったWebアプリです。
よく決まった4人で様々なものを賭けてオンラインゲームをしていたため、
その点数をバランスよくポイント化し、合計を出すアプリを作りました。

Mobile Order System
3ヶ月
Angular
飲食店の大規模なモバイルオーダーシステムのフロント部分を担当しました。
Angularの深い部分にまでは触れられていませんが、SPA、コンポーネント指向など
モダンフロントエンドの雰囲気はそこでなんとなく掴みました。
また、Github Issueをベースにしたフルリモートのチーム開発で、コードレビューやlinter、
CIなどによる品質管理の重要性も学びました。
